## 🚀 Como Instalar e Executar

### 1. Banco de Dados
1. Abra o **phpMyAdmin** (XAMPP).
2. Crie um banco chamado `loja_ecotech`.
3. Execute o arquivo **`db.sql`** (ele já contém `CREATE TABLE` + dados de exemplo).

### 2. Arquivos
1. Copie toda a pasta `loja-ecotech` para dentro de `htdocs` (XAMPP).
2. Inicie o **Apache** e o **MySQL** no XAMPP.

### 3. Acessar
Abra o navegador e digite:  
**`http://localhost/loja-ecotech/index.html`**

## ✨ Funcionalidades

- ✅ Cadastro de produtos (nome, preço, estoque, descrição)  
- ✅ Cadastro de clientes (nome, e-mail, telefone)  
- ✅ Cadastro de pedidos (vinculado a cliente + total)  
- ✅ Listagem em tabelas com dados reais do MySQL  
- ✅ API simples (GET + POST) em PHP  
- ✅ Design responsivo e acessível  
- ✅ Todos os arquivos comentados para estudo
